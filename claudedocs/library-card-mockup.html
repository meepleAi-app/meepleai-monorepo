Design Notes

Aesthetic Direction: Refined tabletop gaming with warm neutrals, subtle textures, and elegant typography.

Navigation Flow:

  • Click on card → Navigate to game personal page (split view: AI Agent + PDF Viewer)
  • Click "Chatta" button → Same behavior (navigate to game page)
  • Click "Azioni" button → Open modal overlay (no navigation, stays on library page)
  • Favorite toggle → In-place action (no navigation)
  • Selection mode → Checkbox selection (no card navigation)

Key Features:

  • Paper grain texture overlay for tactile warmth
  • Soft shadows suggesting physical cards
  • State-coded badges with gentle glow animation
  • Smooth modal entrance with staggered action reveals
  • Mobile-optimized with touch-friendly 44px+ targets
  • Quicksand for headers (warmth) + DM Sans for body (clarity)

Interactions: Card lift on hover, shimmer effect on buttons, heart pop animation for favorites, slide-up modal with backdrop blur.

Accessibility: All actions keyboard navigable, color-blind friendly state coding, sufficient contrast ratios, clear focus states.
