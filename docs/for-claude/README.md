## Conventions

- **No tutorials** here — pure reference for AI/architects.
- Cross-link to code with file paths (`apps/api/src/Api/BoundedContexts/{X}/...`).
- Diagrams: prefer Mermaid in code blocks; static SVG/PNG are auto-generated by `.github/workflows/generate-diagrams.yml` from `.mmd` source (deleted from tree to reduce noise — regen on demand).
- ADRs are append-only. Superseding decisions get a new ADR linking the old one.
