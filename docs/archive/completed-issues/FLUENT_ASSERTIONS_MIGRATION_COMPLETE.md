### What Was Converted
- Assert.Equal/NotEqual → Should().Be/NotBe()
- Assert.True/False → Should().BeTrue/BeFalse()
- Assert.Null/NotNull → Should().BeNull/NotBeNull()
- Assert.Contains/DoesNotContain → Should().Contain/NotContain()
- Assert.Single/Empty → Should().ContainSingle/BeEmpty()
- Assert.ThrowsAsync → Should().ThrowAsync()
- Assert.IsType → Should().BeOfType()
- Assert.InRange → Should().BeInRange()
- Assert.StartsWith/EndsWith → Should().StartWith/EndWith()
- Assert.All → Should().OnlyContain()
- Assert.Collection → Indexed assertions or HaveCount()
- And many more...
