# Typography

MeepleAI uses a carefully curated typography system with two primary font families: **Quicksand** for headings and **Nunito** for body text. Both are friendly, rounded sans-serifs that reflect our playful aesthetic.

## Font Families

### Quicksand (Headings)

<div style={{ fontFamily: 'var(--font-quicksand, system-ui)', marginBottom: '2rem' }}>
  <p style={{ fontSize: '2.25rem', fontWeight: 700 }}>The Quick Brown Fox Jumps Over The Lazy Dog</p>
  <p style={{ fontSize: '1rem', color: 'hsl(var(--muted-foreground))' }}>ABCDEFGHIJKLMNOPQRSTUVWXYZ abcdefghijklmnopqrstuvwxyz 0123456789</p>
</div>

- **Usage**: Headings, titles, display text
- **Weights**: Medium (500), Semibold (600), Bold (700)
- **CSS Variable**: `var(--font-quicksand)` or `var(--font-heading)`

### Nunito (Body)

<div style={{ fontFamily: 'var(--font-nunito, system-ui)', marginBottom: '2rem' }}>
  <p style={{ fontSize: '1.125rem' }}>The Quick Brown Fox Jumps Over The Lazy Dog</p>
  <p style={{ fontSize: '1rem', color: 'hsl(var(--muted-foreground))' }}>ABCDEFGHIJKLMNOPQRSTUVWXYZ abcdefghijklmnopqrstuvwxyz 0123456789</p>
</div>

- **Usage**: Body text, paragraphs, UI labels
- **Weights**: Regular (400), Medium (500), Semibold (600)
- **CSS Variable**: `var(--font-nunito)` or `var(--font-body)`

| Token | Value | Usage |
|-------|-------|-------|
| `--leading-none` | 1 | Headings, single-line |

| `--leading-relaxed` | 1.625 | Long-form reading |
| `--leading-loose` | 2 | Spaced lists |

## Heading Styles

### Recommended Usage

| Level | Size | Weight | Use Case |
|-------|------|--------|----------|
| H1 | 4xl-5xl | Bold | Page titles |
| H2 | 2xl-3xl | Semibold | Section headers |
| H3 | xl-2xl | Semibold | Subsections |
| H4 | lg | Semibold | Card titles |
| H5 | base | Semibold | Labels |
| H6 | sm | Medium | Meta text |

### Example

```tsx
<h1 className="font-quicksand text-4xl font-bold tracking-tight">
  Page Title
</h1>
<h2 className="font-quicksand text-2xl font-semibold">
  Section Header
</h2>
<p className="font-nunito text-base leading-relaxed">
  Body text with comfortable reading line height.
</p>
```

## Best Practices

1. **Use Quicksand for headings only** - It's optimized for display sizes
2. **Nunito for body text** - Better readability at smaller sizes
3. **Maintain hierarchy** - Clear distinction between heading levels
4. **Line length** - Aim for 65-75 characters for body text (`max-w-prose`)
5. **Contrast** - Ensure text meets WCAG AA contrast requirements
