📐 Note di implementazione per DashboardClient.tsx

Logica "new user" per Games const isNewUser = !libraryLoading && gameItems.length === 0 Se isNewUser: mostra CatalogTopGames (da useGames, ordinati per rating, prioritizzati se hasKbVectors). Altrimenti: mostra libreria normale.
Badge KB su MeepleCard Serve un nuovo prop kbStatus: 'indexed' | 'empty' | undefined su MeepleCard, oppure gestire il badge fuori dalla card, wrappando con un overlay relativo. Preferire overlay esterno per non modificare MeepleCard.
Empty CTA per Sessions/Agents Creare EmptyCTA component con props: icon, title, sub, actions[]. Sostituisce l'attuale EmptyState quando si è in dashboard (non in hub page).
Toolkit carousel Dati statici (non da API). Array TOOLKIT_TOOLS con id/icon/name/desc. Ogni card è un Link a /toolkit?tool=dado etc. Non serve HubLayout wrapper per il toolkit nel dashboard.
