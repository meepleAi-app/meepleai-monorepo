ACTION HERO

Game Library Detail Page - Proposta Design A

Brutalist gaming arcade style con focus su azioni immediate, tipografia bold e contrasto elevato. Ispirato ai cabinet arcade e alle UI dei fighting game.

Design Specs

Typography
Bebas Neue + DM Sans
Color Palette
#1a1a1a + #c0ff00 + #ff3366
Style
Brutalist Gaming Arcade
Focus
Immediate Action CTAs

DESKTOP VIEW

Vista desktop principale con hero image full-bleed, stats bar, e azioni primarie verticali.

  • Hero section con status badge
  • 4 stats metrics con icone
  • 3 primary action cards (AI/Checklist/Play)
  • Sticky bottom bar per azioni secondarie
 📱

MOBILE VIEW

Ottimizzazione mobile con stato "Nuovo" e banner tutorial per prima partita.

  • Stats grid 2x2 (mobile-friendly)
  • Banner "Prima Partita!" pulsante
  • Touch-optimized action cards
  • Compact bottom navigation
 📤

STATO "IN PRESTITO"

Vista con gioco prestato: azioni disabilitate e banner promemoria.

  • Status badge rosso "In Prestito"
  • Banner alert con CTA "Ricorda"
  • Azioni primarie disabilitate
  • Regolamento ancora accessibile
 ⚡

REGISTRAZIONE ADAPTIVE

Modal con form adattivo: quick 30s che si espande a dettagliato.

  • Quick mode: 2 campi (vincitore + durata)
  • Detailed mode: +4 campi (giocatori, punteggi, strategie, note)
  • Toggle fluido tra modalità
  • Mode indicator visuale
 💬

AI CHAT

Chat AI context-aware con modalità esperto e conversazione demo.

  • Context badge (esperto vs principiante)
  • Conversazione interattiva funzionante
  • Typing indicator animato
  • Risposte simulate realistiche
