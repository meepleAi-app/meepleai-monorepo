## Quick Reference

### Scope
- **250+ components** across entire frontend
- **75+ pages** (auth, public, chat, admin, misc)
- **9 implementation waves** (Foundation → Testing)
- **18 days estimated** (12-14 working days with parallelization)

### Design Systems
- **Light Mode**: Glass morphism with translucent cards, backdrop blur, floating orbs
- **Dark Mode**: Professional with deep backgrounds, glowing accents, high contrast

### Key Decisions
1. **Theme**: Hybrid (next-themes + CSS variables)
2. **Migration**: Big bang (all components at once)
3. **Animation**: CSS for simple, Framer Motion for complex
4. **Testing**: Comprehensive (all tests updated, visual regression, E2E)
5. **Automation**: Morphllm for 95% time savings
6. **Branches**: Parallel on main-dev (admin) + frontend-dev (public)
