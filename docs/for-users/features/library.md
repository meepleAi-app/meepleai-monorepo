# Libreria personale

> **Cos'è in una frase**: la tua collezione di giochi da tavolo gestita su MeepleAI, con scaffali, wishlist e statistiche di gioco.

---

## In breve

| | |
|---|---|
| **A cosa serve** | Tenere traccia di cosa possiedi, cosa vuoi, cosa hai giocato |
| **Per chi** | Tutti gli utenti |
| **Vista principale** | Pagina **Libreria** (`/library`) |
| **Sincronizzazione** | Cloud — vedi gli stessi dati da web e mobile |

---

## Cosa contiene la libreria

### Scaffali principali

| Scaffale | Cosa ci finisce |
|----------|-----------------|
| **Posseduti** | Giochi che hai fisicamente in casa |
| **Wishlist** | Vorrei comprarli |
| **Vorrei provarli** | Curiosità senza intento d'acquisto |
| **Prestati** | Hai prestato a un amico (segna nome + data) |
| **In prestito da** | Hai preso in prestito (memo per restituire) |
| **Archiviati** | Venduti o regalati (storia, non più nella collezione attiva) |

Un gioco può essere **in più scaffali** — utile per "Posseduto + Wishlist espansione".

### Tag personali

Aggiungi tag liberi: `#leggero`, `#party`, `#serata-coppia`, `#weekend-lungo`. Filtra/ordina per tag in qualunque scaffale.

### Note private

Su ogni gioco puoi annotare: punteggio personale, varianti casalinghe, regole house, prezzo pagato, dove l'hai comprato.

---

## Statistiche & storico partite

Quando finisci una sessione (vedi *Sessions* — pagina futura), MeepleAI traccia:

- Numero partite giocate
- Win rate (se sei tracker)
- Durata media partita
- Compagni di gioco frequenti
- Ultima partita giocata

Visibile da: **Libreria** → click sul gioco → tab **Statistiche**.

---

## Visualizzazioni

| Vista | Quando usarla |
|-------|---------------|
| **Griglia** (default) | Vedere tutte le copertine — bello visivamente |
| **Lista** | Cercare velocemente per titolo, durata, peso |
| **Compatta** | Scrollare collezioni grandi (>200 giochi) |
| **Dettaglio** | Click su un gioco → scheda completa |

---

## Filtri e ordinamento

Filtra per: scaffale, tag, n° giocatori, durata, peso, anno, ultima partita giocata.
Ordina per: titolo, anno, voto BGG, voto personale, ultima partita, data aggiunta.

Esempio combinato: *"Posseduti, 2-3 giocatori, 30-60min, ordinati per ultima partita giocata"* → cosa potresti tirare fuori stasera per una serata di coppia.

---

## Condivisione

### Condividi un gioco singolo

Click sul gioco → **Condividi** → genera link pubblico read-only (non richiede account).

### Condividi tutta la collezione

**Libreria** → **Esporta** → link pubblico o file (CSV/JSON). Utile per scambi tra appassionati.

---

## Sincronizzazione mobile

L'app mobile (PWA) usa lo stesso account: la libreria è **identica** su web e mobile. Modifiche offline si sincronizzano al rientro online.

---

## Limiti noti

- ⚠️ **Max 5000 giochi** per libreria (limite anti-abuse, raggiungibile solo da grossi collezionisti).
- ⚠️ **Foto custom della copia** (back of box, condizioni) non ancora supportate — solo copertina ufficiale BGG.
- ❌ **Quotazione/valore di mercato** non tracciata (puoi annotare in nota privata).
