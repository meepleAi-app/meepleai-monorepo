Nuove Props (chatSession)

chatStatus?: 'active' | 'waiting' | 'archived' | 'closed'

chatAgent?: {
  name: string
  avatarUrl?: string
  modelName: string
}

chatGame?: {
  name: string
  id: string
}

chatStats?: {
  messageCount: number
  duration?: string
  lastMessageAt: string
  rulesCount?: number
}

chatPreview?: {
  messages: Array<{
    sender: 'user' | 'agent'
    text: string
  }>
  isTyping?: boolean
}

unreadCount?: number

ChatStatusBadge

ChatMessagePreview
Bubble-style preview con sender indicator (Tu/AI).
Typing indicator animato per stato "active"

ChatUnreadBadge
Badge numerico circolare, chat-color bg.
Highlight su list variant (bg tinted)

ChatGameContext
Chip con dot arancione (game entity color).
Link al gioco associato

ChatStatsDisplay
Message count, durata, rules cited.
Layout: horizontal | vertical (come AgentStatsDisplay)
