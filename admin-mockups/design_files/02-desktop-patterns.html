📜 Storico
A · Split view

Sidebar stretta + lista selezionabile + pannello dettaglio. Alla Linear/Mail/Superhuman.

✓ Best for: browsing veloce, triage, scan di molte entità dello stesso tipo.
B · Sidebar + Drawer

Nav gerarchica + documento principale + drawer laterale per entità correlate. Alla Notion/Slack.

✓ Best for: lavoro profondo, esplorazione relazioni, multi-context.
C · Tabs persistenti

Hero fisso + tab orizzontali che restano visibili. Alla Vercel/GitHub/Stripe.

✓ Best for: navigazione tra aspetti di una singola entità, dashboard monografica.
