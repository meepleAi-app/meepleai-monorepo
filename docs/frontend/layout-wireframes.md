## 8. Accessibility Annotations

### 8.1 Focus Order

```
Mobile:
1. Hamburger menu
2. Logo (skip link to content)
3. Search
4. Profile
5. Content (main)
6. FAB
7. Breadcrumb
8. ActionBar items (left to right)
```

### 8.2 ARIA Labels

```
FAB:
  aria-label: "{context-specific action}"
  e.g., "Inizia nuova sessione" / "Aggiungi gioco"

ActionBar item:
  aria-label: "{action name}"
  role: "button"

Quick menu:
  role: "menu"
  aria-labelledby: FAB id

Breadcrumb:
  role: "navigation"
  aria-label: "Contesto corrente"
```
